# 🔑 OPEN Password

**OPEN Password** is a smart, privacy-focused browser extension that automatically generates, detects, saves, and autofills passwords—designed for a seamless and secure browsing experience.

It is part of the **OPEN Security Ecosystem**, working alongside **OPEN OTP** to provide a complete authentication solution.

---

## 🌐 OPEN Ecosystem

The OPEN ecosystem simplifies secure login:

### 🔑 OPEN Password (this project)

* Generate and autofill passwords
* Detect login/signup forms automatically
* Securely store credentials

### 🔐 OPEN OTP

* Generate 6-digit 2FA codes
* Support multiple accounts
* Enhance login security

👉 Together, they provide:

* Autofill + OTP in one flow
* Faster and more secure logins
* A modern authentication experience

## 🚀 Advanced Features

### 🧠 Smart Password Auto-Detection

* Runs on every website via content script
* Detects password fields automatically
* Identifies context (Sign In / Sign Up)
* Displays floating **“Suggest Strong Password”** button

### ⚡ One-Click Auto-Fill

* Generates a secure 20-character password instantly
* Autofills password fields
* Automatically fills **confirm password** fields

---

### 💾 Intelligent Auto-Saving

#### 🔄 Background Sync (Pending Saves)

* Passwords generated on websites are temporarily cached
* When Vault is unlocked:

  * Automatically encrypted
  * Saved under the correct domain

#### ⚡ Instant Popup Saving

* Save directly from extension popup
* Detects current website using activeTab
* Saves without requiring manual naming

---

### 🔐 Vault System

#### 🎨 Premium Glassmorphism UI

* Frosted glass effect (`backdrop-filter`)
* Translucent dark design
* Smooth hover animations

#### 🔍 Real-Time Search

* Instantly filter saved credentials
* Dynamic search while typing

#### ✨ Staggered Animations

* Password cards animate into view

## 🧠 How It Works

OPEN Password enhances your browsing by:

* Detecting login/signup forms automatically
* Suggesting strong passwords

## 🔐 Security

* All data stored **locally on the user's device**
* Encryption using **Web Crypto API (AES)**
* No cloud sync or external servers
* Domain-based storage for isolation

> ⚠️ Always protect your device and backup important credentials

---

## 🛠️ Tech Stack

* JavaScript (Vanilla)
* HTML5 & CSS3
* Web Crypto API (local encryption)
* Chrome Extension APIs (storage, scripting, activeTab)

## 📦 Installation

### 🔧 Load Extension (Developer Mode)

1. Clone the repository:

   ```bash
   git clone https://github.com/yourusername/open-password.git
   ```

2. Open Chrome:

   ```
   chrome://extensions/
   ```

3. Enable **Developer Mode**

4. Click **Load unpacked**

5. Select the project folder

## 📱 Usage
